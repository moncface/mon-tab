# Mon tab

> Omnibox-based developer CLI for Chrome. 50 commands across 15 categories.

Lightweight Chrome extension that turns the browser address bar into a developer command line. No API dependencies. No data collection. Zero external requests.

## Install
Chrome Web Store: [Mon tab](https://chrome.google.com/webstore/detail/mon-tab/)

## Core Features
- [Omnibox CLI]: Type commands directly in Chrome's address bar
- [Pulldown GUI]: 5-layer entry system eliminating command memorization
- [WCAG Tools]: Accessibility audit commands
- [Scientific Calculator]: Math commands in the omnibox
